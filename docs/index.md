# MiniML Documentation

MiniML is a small machine learning library built with NumPy. It is meant for learning how common models work, not for replacing production libraries such as scikit-learn.

## What Is Included

- Linear regression
- Ridge, Lasso, and Elastic Net regression
- Logistic regression
- K-nearest neighbors
- Basic metrics
- Train/test splitting, cross validation, and grid search
- Simple preprocessing tools

## Where To Start

1. Install the package from the project root.
2. Read the getting started tutorial.
3. Try the examples in `examples/`.
4. Open the math pages when you want to understand the objective and gradient behind each model.
