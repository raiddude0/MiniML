# MiniML
MiniML is a NumPy-based machine learning library implemented from scratch.
It includes linear regression, ridge regression, logistic regression,
preprocessing utilities, evaluation metrics, cross-validation, and model
selection tools.

## Why This Project Exists
## Features
    -linear regression trained with batch gradient descent
    -Ridge, Lasso, and Elastic Net regularization (WIP)
    -Logistic regression for binary classification
    -Standard scaling and train/test splitting
    -Regression metrics: MSE, MAE, RMSE, R2
    -Classification metrics: accuracy, precision, recall, F1, confusion matrix (WIP)
    -Cross-validation and grid search (WIP)
    -Loss-history tracking for optimization diagnostics
    -Unit tests for numerical correctness
    -Comparisons against scikit-learn implementations (WIP)
